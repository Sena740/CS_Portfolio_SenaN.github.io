Rock-Paper-Scissors Game

For my ISP, I wanted to touch on something I not yet learned, which was untilizing the computer camera. Through this I learned how to use Media Pipe, which required me to use an older version of Python leading me to some problem I had to solve. At the end I managed to create a hand gesture detector to be used in my simple Rock-Paper-Scissors Game.

Tower Defence Game

For our group project, we dicided to create a tower defense game, using pre-existing designs from a game called Genshin Impact. Using coding methods learned in class, and combining it with some tutorials here and there, my team (Sena, Jayden, Jenny) were able to create this game, but was not perfect. We were unable to fix an issue where placing more than one tower would cause an error with the shooting system (Check second video for more detail). Furthermore, I personally think we had our goals set too high since we failed to complete simple tasks along the creation period. A good experience overall.
